# 🎤 RasPet 발표 대본 (약 3분)

> 분량 기준: 천천히 읽어 약 3분(≈480단어). 괄호 `( )`는 동작/화면 지시.

---

**[오프닝 — 약 20초]**

안녕하세요, 임베디드시스템 프로젝트 **RasPet**을 발표할 202312689 박영주입니다.
RasPet은 라즈베리파이 5 위에서, 손바닥만 한 OLED 화면으로 **나만의 펫을 키우는 휴대형 게임기**입니다.
어릴 때 하던 다마고치에, 능력치를 키워 결말이 갈리는 프린세스 메이커를 합쳤다고 보시면 됩니다.

**[개요 — 약 25초]**

(화면을 보여주며) 게임기를 켜면 OLED에 캐릭터가 등장합니다.
시간이 지나면 배가 고프고 스트레스를 받죠. 사용자는 미니게임으로 재화를 벌어 펫을 돌보고,
체력·지력·매력 같은 능력치를 키웁니다. 그리고 **최종 능력치 조합에 따라 서로 다른 엔딩**에 도달합니다.
카메라로 하는 컴퓨터 비전과, 오목을 두는 게임 AI를 **하나의 작품 안에서** 다룬 것이 핵심입니다.

**[시스템 구성 — 약 25초]**

하드웨어는 라즈베리파이 5에 OLED, 카메라, 초음파 센서, 조이스틱, 키패드, 조도·온도 센서,
LED와 7세그먼트, 부저를 붙였습니다. 핀이 부족해서 ADC 칩 MCP3008으로 아날로그 입력을 받고,
LED와 7세그먼트, 키패드와 버튼은 **핀을 공유**하도록 설계해 한정된 40핀 안에 모두 넣었습니다.
소프트웨어는 Python과 Pygame 게임 루프 위에 OpenCV 비전, 오목 AI, gpiozero 하드웨어 제어를 얹었습니다.

**[핵심 기능 데모 — 약 70초]**

(가위바위보 실행) 첫 번째 필수 기능은 **가위바위보**입니다. 카메라로 손을 인식해서 가위·바위·보를 판정합니다.
원래 MediaPipe를 쓰려 했지만 라즈베리파이 환경에서 설치가 안 돼서, **OpenCV로 손가락 수와 각도를 분석**하는
방식으로 직접 구현했습니다.

(오목 실행) 두 번째는 **오목**입니다. Minimax에 알파-베타 가지치기를 적용한 AI와 대결합니다.

여기에 더해 초음파 센서로 거리를 재서 캐릭터가 점프하는 **플래피버드식 게임**,
OpenCV로 색을 맞히는 **색깔 찾기**, 4×4 키패드로 하는 **두더지 잡기**까지 미니게임을 넣었습니다.

(메인 화면) 특히 환경 반응이 재미 포인트인데요. **조도 센서**가 어두워지면 펫이 잠을 자고,
**온도 센서 BMP180**으로 더우면 땀을 흘리고 추우면 눈송이 표정을 짓습니다.
이때 **LED**가 온도에 따라 빨강·초록·파랑으로 같이 바뀌고, 7세그먼트에는 현재 온도가 표시됩니다.

**[기술 포인트 — 약 20초]**

개발하면서 가장 신경 쓴 부분은 **안정성**입니다.
모든 하드웨어를 추상화 계층 뒤에 두고, 장치가 없으면 자동으로 더미로 동작하게 만들어서
라즈베리파이가 없는 PC에서도 똑같이 돌아갑니다. 덕분에 테스트 97개와 GitHub Actions CI로
코드를 계속 검증하면서 개발할 수 있었습니다.

**[마무리 — 약 10초]**

정리하면, RasPet은 **카메라 비전·다중 센서·게임 AI를 하나의 임베디드 게임기로 통합**한 프로젝트입니다.
들어주셔서 감사합니다. 질문 받겠습니다.
